## 내용

### 게임플레이

- 세션마다 트랙 플레이리스트를 무작위로 셔플합니다.
- 라운드별 타이머와 라운드 수를 설정할 수 있습니다.
- 정답 판정 시 공백과 쉼표를 제거해 비교합니다.
- `End` 키로 스킵할 수 있으며, 종료 화면에서 최종 점수를 표시합니다.

### 아키텍처

- 멀티스레드 설계: 타이머, 대기 카운트다운, 앨범 페이드, 버튼 이펙트를 각각 전용 스레드로 실행합니다.
- 세마포어 기반으로 스레드 간 상태를 동기화합니다.

### UI

- Java Swing 기반, 컴포넌트 스타일을 전부 커스텀했습니다.
- 호버 효과가 있는 커스텀 슬라이더 UI
- 정답 공개 시 앨범 아트 페이드 인 애니메이션
- 로그 스케일 볼륨 슬라이더를 마스터 게인에 매핑
- 인라인 HTML로 한글/라틴 폰트를 혼합 렌더링

### 설정

- 타이머와 라운드 수를 `settings.ini`에 저장합니다.
- 시작 시 로드하고, 슬라이더 변경 시 실시간으로 저장합니다.
